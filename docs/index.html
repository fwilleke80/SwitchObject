About Switch Object

What is it?

Sometimes you want to create different versions of an object or scene. Like a room with different sets of curtains, a truck with different trailers or just alternative light setups for a scene (e.g. day, evening, night).

Of course, you could do this with XPresso, creating User Data entries for all your options, connecting lots of nodes to check for the parameters and the visibility values of all lots of hierarchy branches... and if you added another curtain to your room or another trailer to your truck, you would have to worry about wiring the new objects, too, and updating all the User Data entries.

Switch Object makes it really easy for you: No Xpresso, no User Data, just straight forward switching of object groups.

Requirements

This plugin works on Windows (32/64 Bit) and Intel Mac with OSX 10.5 or higher.

CINEMA 4D R11.5 or higher is required.

Installation

Just unzip the archive and place the folder "SwitchObject" inside the folder "CINEMA 4D R12/plugins".

When you start CINEMA 4D the next time, the Switch Object will be available from the Plugins menu.

Usage

There are two ways of using the Switch Object:

1. Like a Null Object

This is the simplest way of using the Switch Object. Just use a Switch Object instead of a Null Object to group all the objects that should be available for selection.

- Switch Object
  + Cube
  + Torus
  + Sphere

Now, if you select the Switch Object and take a look into the Attribute Manager. The parameter "Show" automatically offers you all child objects.

Choose any item from the list to make it visible, while hiding all the others.

2. Like a remote control

Maybe your hierarchy is more complex, and you don't want to scatter the Switch Objects over the complete scene? No problem, the Switch Object is the answer again.

This way you can put your Switch Objects into a central location of your hierarchy, and let them 'remote control' other objects in the scene.

Group the objects you want to make available for selection into a Null Object, just as normal. Now create a Switch Object and select it. Drag & Drop the Null Object into the "Remote Link" field in the Switch Object's attributes.

  + Torus
  + Sphere

If you take a look into the Switch Object's attributes now, the parameter "Show" offers all child objects of the linked Null Object.

Choose any item from the list to make it visible, while hiding all the others.
